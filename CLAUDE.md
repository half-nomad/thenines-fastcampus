## ⚠️ 핵심 규칙

### 2. 문서 참조 순서
```
TODO.md → PLAN.md → DESIGN_GUIDE.md → PRD.md
```

- 작업 완료: `[~]` → `[x]`
- Phase 완료: 작업 로그 기록
